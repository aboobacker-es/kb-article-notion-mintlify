For a candidate to be able to retake your test (which is submitting another attempt after the most recent one), you can either [📄 Reinviting Candidates to a Test](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/a7b4d9b2-f6b2-4b57-826b-30061839be41)to the same test (when you do so, a candidate's test status initializes to "Invited").

<Note>
**Note**: When Candidates try to access a Test for a reattempt, sometimes, a browser-level cache may disallow them from logging in to the test. To overcome this issue, it is recommended that your candidates access the Test link using a new browser window.
</Note>

To re-invite candidates to a test, you can select one or more candidates from the **Candidates** tab in your test and click on **Invite Selected** as shown below. For more information on re-inviting candidates, refer to the [📄 Reinviting Candidates to a Test](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/a7b4d9b2-f6b2-4b57-826b-30061839be41).

<Note>
**Note**: When you re-invite a candidate, the most recent attempt by the candidate is archived. A candidate can be re-invited as many times as needed.
</Note>
