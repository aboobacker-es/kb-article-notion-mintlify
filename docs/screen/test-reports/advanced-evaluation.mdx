This feature is part of the AI Add-on. For more information, see [📄 HackerRank AI Add-on](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/8704fb40-5ded-4188-b5a6-cfd653d0f5c9).

HackerRank Advanced Evaluation provides richer signals to help you assess a candidate’s performance more comprehensively. It offers insight into code quality, efficiency, and AI fluency across the candidate journey.

As AI becomes an integral part of software development, engineering teams are redefining how they evaluate NextGen Developers. Correctness alone is no longer enough. Modern engineering teams look for developers who efficiently reach the correct solution, write clean code, and show sound judgment, especially when collaborating with AI tools.

Advanced Evaluation helps you assess the skills that matter most in today’s coding environments.

### Available evaluation signals

The following evaluation signals are available:

- **Code Quality: **Evaluate how clean, maintainable, and well-structured a candidate’s code is. For more information, see [📄 Code Quality Evaluation](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/753a2490-35b2-4071-b8a4-6df58da25255).
- **Optimality: **Evaluate how efficient a candidate’s solution is in terms of time and space complexity. For more information, see [📄 Optimality ](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/9f70e231-12d5-47fd-b5c2-7285457cc8a8).
- **AI Fluency: **Evaluate how the candidate interacts with the AI assistant during a test. For more information, see [📄 AI Fluency Evaluation](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/5075d153-4ed1-4bf5-bea2-8fbdd0a4643f).
- **Automated Code Review Scoring: **Automatically evaluates a candidate’s code review comments against expert examples. For more information, see [📄 Automated Code Review Scoring](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/5e08aa19-c6e4-4e23-a07d-4eff4641bf33).

### Enable advanced evaluation

To enable advanced evaluation:

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to **Settings**.
1. Select** Test Settings** from the left panel under the **Company** section.
1. In the **Test Evaluation** section, toggle **Enable Advanced Evaluation** to activate all of the additional signals in your reports.
