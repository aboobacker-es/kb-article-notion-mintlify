HackerRank Insights, powered by Highcharts, provides powerful analytics to evaluate candidate performance across tests. The interactive graphs enable you to explore trends in scores, assess question difficulty, and monitor overall test completion rates.

#### Key Features:

- **Score Distribution Graphs**: Analyze candidate performance by viewing score distributions across a test
- **Question-Level Insights**: Compare the time taken per question with accuracy rates to optimize question quality.
- **Candidate Performance Graphs**: Track individual candidate scores and pacing relative to the average performance of all test-takers.

### Viewing Test Insights

1. Navigate to **Tests **and select a test.
1. Click on **Insights** to view the test insights page.

#### Test Details

The **Test Details** section provides essential information, including details like test name, test ID, test creation, last updated dates, test owner name and email, and allotted duration. These details offer insights into the test's development history and structure.

#### Test Activities

**Candidate Status:** The **Candidate Status** pie chart shows the distribution of candidates who have passed, failed, or are yet to be evaluated.

**Additional Widgets:** On the right side, the following widgets are available:

- **Total Invited**: Displays the total number of candidates invited to take the test.
- **Total Attempted**: Shows the total number of candidates who have submitted the test.
- **Attempt Rate**: Indicates the percentage of candidates who attempted the test out of the total invited.

The test level insights dashboard will contain the following details:

- Distribution of Candidate Score
- Median Score
- Median Attempt Time (Minutes)
- Median Time to Full Score (Minutes)
- Test Feedback

#### Question Level Insights

For each question in the test, the graphs display the median and average scores achieved by candidates. Use these insights to identify which questions yielded higher or lower scores. The question level insights dashboard will contain the following details:

- Median Score per Question
- Average Score per Question
- Median Time per Question
- Average Time per Question

#### Performance Funnel - All Candidates

**Test Invited: **The total number of candidates invited to the test, including those who may or may not have attempted it.

**Test Link Clicked: **The total number of candidates who clicked the link to take the test.

**Test Attempted: **The total number of candidates who submitted the test.

**Test Attempted at least One Question:** The total number of candidates who attempted at least one question.

#### Test Attempt Insights

**Number of Attempts Over Time: **This graph displays the number of test attempts based on the start date of each attempt.

**Attempt Duration Distribution (%):** This graph shows the distribution of test attempts across different attempt duration percentage categories.
