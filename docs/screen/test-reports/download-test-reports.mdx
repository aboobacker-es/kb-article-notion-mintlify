HackerRank test reports provide a detailed view of a candidate’s performance and scores in a test. After candidates complete a test, you can view their detailed performance reports and download them in PDF or Excel formats.

You can use the downloaded reports to share with external evaluators or recruiters or to maintain internal performance records.

<Note>
**Note:** You can download test reports only for candidates who have completed the test.
</Note>

## PDF test reports

You can download individual PDF reports for each candidate. Each PDF report replicates the detailed test report and includes the following information:

<Frame>
  <img src="/images/download-test-reports/2753916c248ca99f.png" alt="image" />
</Frame>

- Score
- Candidate information
- Skill distribution
- Tag distribution
- Questions and candidate answers

For more information about detailed test report, see [📄 Viewing a Candidate's Detailed Test report](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/0932710c-bad5-4bab-9878-a1a9b7163d14).

If you use the new version of the summary report, the PDF also includes:

<Frame>
  <img src="/images/download-test-reports/816299f6e2647bf5.gif" alt="image" />
</Frame>

- Code quality
- Integrity issues
- Performance summary
- Integrity summary
- Questions and answers with code quality details (If available)

<Note>
**Note:** Although the PDF report includes the candidate's detailed answers to the questions, use the Detailed Report view in HackerRank for a thorough evaluation. The detailed report provides features such as **c**ode playback, compilation, rendering, and download.
</Note>

## Excel test reports

The Excel test report contains the performance report for all candidates in a single file. You can customize the report in [Report Settings](https://hackerrank-knowledge-base.help.usepylon.com/articles/5213013268-report-settings#accessing-report-settings-1) to choose which candidate performance data to include when downloading it in Excel format.

## Downloading test reports

To download test reports in PDF or Excel format:

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to the **Tests** tab.
1. Open the test whose reports you want to download.
1. Go to the **Candidates** tab. The **Status** column indicates the test status of each candidate.
1. In the left pane, expand **Candidate Status** and select **Completed**.

  The right pane displays all candidates who have completed the test and are marked as `To Evaluate`, `Passed`, or `Failed`.
