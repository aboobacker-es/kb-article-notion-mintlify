You can set scoring thresholds and apply uniform scoring logic for multiple-choice questions (MCQs) to ensure consistent evaluation across all candidates.

### Prerequisites

You must have already created a test.

### Modifying evaluation settings

To modify evaluation settings:

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to the **Tests** tab.
1. Open the test you want to update.
1. Go to **Settings \> Evaluation.**
