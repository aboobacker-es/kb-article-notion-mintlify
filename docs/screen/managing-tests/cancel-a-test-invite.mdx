HackerRank for Work allows you to cancel a test invitation that you have already sent to a candidate.

### Prerequisites

- You must have created the test.
- You must have already sent the invitation to the candidate.

### Canceling a test invitation

To cancel a test invitation:

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to the **Tests** tab.
1. Open the test from which you want to cancel a candidate's invitation.
1. Go to the **Candidates** tab.

  <Frame>
    <img src="/images/cancel-a-test-invite/56d592a93009b412.bin" alt="image" />
  </Frame>
1. Select the checkbox next to the candidate’s email address.

<Note>
**Note:** You can select one or more candidates.
</Note>

1. Click **More** \> **Cancel Invite**.

  <Frame>
    <img src="/images/cancel-a-test-invite/5a7da5955b540971.bin" alt="image" />
  </Frame>

The candidate(s) will be removed from the test’s invitation list. The number of canceled invitations is credited back to your company’s account.

### Candidate experience

Canceled candidates can no longer access the test. If they attempt to start the test, an error message is displayed.

<Frame>
  <img src="/images/cancel-a-test-invite/45e03c712aeebee0.bin" alt="image" />
</Frame>

For more details about inviting candidates, see [📄 Invite Candidates to a Test](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/7911bb8a-2966-4ca7-b9ae-956dfbc6fb99).
