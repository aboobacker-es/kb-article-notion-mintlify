HackerRank for Work allows admins to revoke test access from specific users or teams.

### Prerequisites

- You must be the test owner or have **Editor** access.
- The test must already be shared with the users or teams using the **Share** option.

### Revoking test access

To revoke access to a test for a specific user or team:

1. Log in to your HackerRank for Work account using your credentials.
1. Go to the **Tests** tab.
1. Open the test from which you want to revoke access.
1. Click **Share.**

  <Frame>
    <img src="/images/revoke-test-access/d2180a15401c8851.bin" alt="image" />
  </Frame>
1. In the **Share Test** dialog box, locate the user or team.
1. Select the role drop-down next to their name.
1. Select **Remove User**.

  <Frame>
    <img src="/images/revoke-test-access/34b864d8191e7d93.bin" alt="image" />
  </Frame>
1. In the confirmation dialog box, click **Confirm**.

The user or team is immediately removed and will no longer have access to the test.
