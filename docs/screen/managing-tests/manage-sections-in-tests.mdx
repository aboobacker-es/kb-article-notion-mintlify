Sections help you organize test questions by topic or skill. You can also include code repositories as separate sections to assess real-world development skills. You can set time limits for each section and control how questions appear to candidates. You can configure multiple sections in a test to create a more focused and adaptive assessment experience.

## Prerequisite

You must have created a test in your HackerRank account.

## Add a section to a test

To add a new section:

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to the **Tests** tab.
1. Open the test you want to edit.
1. Go to the **Questions** tab.
1. Select **Add New **next to **Sections**.

You can add, try, replace, or remove questions within a section to control what candidates see and how the questions are presented.

To add a question to a section :

1. Select the section you want to update. The questions in the section appear.

#### Try a question

To try a question in a section:

<Frame>
  <img src="/images/manage-sections-in-tests/8fe5c7d34ca70b7e.gif" alt="image" />
</Frame>

1. Click the more options menu (**⋮**) next to the question.
1. Select **Try question**. A new window opens showing how the question appears to the candidate.

#### Replace a question

To replace a question in a section:

<Frame>
  <img src="/images/manage-sections-in-tests/47fa534dcc08f07a.gif" alt="image" />
</Frame>

1. Click the more options menu (**⋮**) next to the question.
1. Select **Replace**. The Library opens with filters applied based on the original question’s type, skill, and difficulty.
1. Click **Replace** next to the question you want to use.
1. In the confirmation modal, compare the existing and new questions (name, skill, type, score, and duration).
1. Click **Confirm**.

#### Remove a question

To remove a question in a section:

<Frame>
  <img src="/images/manage-sections-in-tests/e8c95fbed6de7972.gif" alt="image" />
</Frame>

1. Click the more options menu (**⋮**) next to the question.
1. Select **Remove**.
1. In the confirmation dialog, click **Delete** to confirm.

## Add a code repository section to a test

Code repository section allow you to assess a candidate’s ability to work with real-world codebases. Each repository includes the following task types that you can choose from:

- Feature Building
- Bug Fixing
- Code Review

Currently, code repositories are available for the following roles:

- Frontend Developer (React)
- Senior Frontend Developer (React)
- Backend Developer (Node.js)
- Senior Backend Developer (Node.js)

You can add one or more repositories in a single test to evaluate different skill areas. You can also add code repositories alongside other skills and question types in the same test.

To add a code repository section:

<Frame>
  <img src="/images/manage-sections-in-tests/a2635ee7fed4ea02.bin" alt="image" />
</Frame>

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to the **Tests** tab.
1. Open the test you want to edit.
1. Go to the **Questions** tab.
1. Select **Add New **next to **Sections**.
1. Select **Add Manually**. A new section labeled **Untitled** appears at the bottom of the list.

  **Note: **To rename the section, see** **[Configure section settings](https://support.hackerrank.com/articles/5546852679-section-based-testing#configure-section-settings-6).
1. Click **Add from Library** to open the question library.
1. Use the **Type** filter and select **Bug Fix**, **Feature, **or **Code Review** to display only code repository questions.
1. Click **+ Add** next to the code repositories you want to include.

<Note>
**Note: **To add a code repository tasks to a code repository section, see [Add a code repository task](https://support.hackerrank.com/articles/5546852679-section-based-testing#add-a-code-repository-task-35).
</Note>

### Add a code repository task

To add a code repository task to a code repository section:

<Frame>
  <img src="/images/manage-sections-in-tests/3d82ba2054986a90.bin" alt="image" />
</Frame>

1. Select the code repository section you want to update.
1. Click **Add from Repository** to open the question library.
1. Select the checkbox next to the question you want to include.
1. Click **Add Task**.
