Screen is a technical skills assessment platform that evaluates candidates through coding tests.

Recruiters can create and send tests to candidates, who complete them remotely in a proctored coding environment. The platform automatically scores objective questions, such as coding challenges, using test cases, and provides candidate reports with scores, code submissions, and integrity signals, including AI plagiarism detection and proctoring data.

## Accessing Tests

To access the Tests:

1. Log in to your **HackerRank for Work** account using your credentials.
1. Go to the **Tests** tab.

  <Frame>
    <img src="/images/introduction-to-screen/27df88db9f3b3556.png" alt="image" />
  </Frame>

The Tests tab contains three sub-tabs:

<Frame>
  <img src="/images/introduction-to-screen/5e714aafa6eac2f2.png" alt="image" />
</Frame>

- **Active Tests**: Displays tests that are in draft or currently being attempted.
- **Archived Tests**: Displays archived tests.
- **Starred Tests**: Displays tests marked as starred.

Each test includes details such as the title, role, number of questions, test duration, test creator, date created, labels, number of candidates who have not attempted, number of candidates who have completed, number of candidates to be evaluated, and the **Invite** icon.
