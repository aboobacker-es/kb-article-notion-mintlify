HackerRank is a technology hiring platform designed to help companies efficiently hire skilled developers in a remote-first world. It enables objective evaluation of tech talent at every stage of the recruiting process. This article introduces the key features of HackerRank Screen to help you get started.

#### HackerRank Library

The HackerRank Library is a centralized repository containing pre-built questions of various complexities and types, covering multiple programming languages. This feature helps you create Tests aligned with your hiring requirements, enabling skill-based and experience-driven candidate assessments.

The library includes:

- Ready-made questions curated by HackerRank to match job roles.
- Categorization by language, complexity, and skill type.
- The ability to create custom questions is stored in the **My Company Questions** section.
- Options to manage shared questions among team members.

#### Tests

HackerRank allows you to create Tests to evaluate candidates’ skills for specific roles or custom requirements. Follow these steps to create a Test:

1. Go to the **Tests** page and click **Create Test**.
1. Choose an applicable role to create a test.
1. Add questions from the Library or your custom question repository using the **Add from Library** option.
1. Filter questions by role, skill, difficulty, and more to match the Test's requirements.

Refer to:

- [Creating Certified Assessments](https://support.hackerrank.com/articles/1649328687-hackerrank-certified-assessments)
- [Creating a Custom Test](https://support.hackerrank.com/articles/2031393332-creating-a-custom-test)

#### Test Settings

Test settings allow you to configure options such as:

- Candidate access
- Test scoring
- Question display
- Time accommodations

To modify settings:

1. Navigate to the **Tests** page and select the desired Test.
1. Click **Settings** and adjust options from the tabs on the left panel.

Refer to related articles for specific configurations:

- [Modifying Candidate Settings for Tests](https://support.hackerrank.com/articles/6890796492-modifying-candidate-settings-for-tests)
- [Proctoring HackerRank Tests](https://support.hackerrank.com/articles/1079706165-proctoring-hackerrank-tests)

#### Inviting Candidates

Invite candidates to a Test via email. To send invitations:

1. Publish the Test.
1. Click **Invite** and enter candidate email addresses or upload a CSV file.
1. Customize the email using templates or saved formats.

<Note>
**Note**: Recruiters must have a valid license to send invitations.
</Note>

#### Test Insights and Reports

#### Test Insights

The **Test Insights** page provides performance analytics to refine your hiring process, including trends and effectiveness metrics.

#### Candidate Reports

Access detailed reports on candidate performance, including code quality and plagiarism detection. Reports can be downloaded as PDFs or Excel files for external sharing.

For more details:

- [Viewing Detailed Test Reports](https://support.hackerrank.com/articles/8263794320-viewing-a-candidate%27s-detailed-test-report)

#### HackerRank Interviews

HackerRank Interviews offers a real-time, interactive platform for conducting technical assessments. Interviews replicate an in-person experience while enabling collaborative problem-solving.

Explore additional resources:
