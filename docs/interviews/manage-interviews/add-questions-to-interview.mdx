You can add existing questions from the HackerRank Library, your company library, or by creating custom questions during an interview session.

## Prerequisite

You must have joined an interview session.

## Adding questions to an interview

You can add questions to an interview using any of the following methods:

- [Question Tiles](https://support.hackerrank.com/articles/6761309830-adding-questions-to-interviews#using-question-tiles-6)
- [Question Library](https://support.hackerrank.com/articles/6761309830-adding-questions-to-interviews#using-question-library-11)
- [Templates](https://support.hackerrank.com/articles/6761309830-adding-questions-to-interviews#using-templates-14)
- [Code Repositories](https://support.hackerrank.com/articles/6761309830-adding-questions-to-interviews#using-code-repositories)
- [Screen Tests](https://support.hackerrank.com/articles/6761309830-adding-questions-to-interviews#import-questions-from-screen-test-27)

### Using Question Tiles

When you start an interview, the **Start Your Interview** page displays tiles for different question types.

To add a question:

1. Select a question type tile from the **Start Your Interview** page.
1. Follow the instructions for the selected question type.

### Using Templates

You can access the pre-built template directory shared by your team.

To import a question from templates:

<Frame>
  <img src="/images/add-questions-to-interview/296ef5eab43d56c2.gif" alt="image" />
</Frame>

1. Select **Templates** from the left panel.
1. Select the template you want to use.
1. Click **Use**.

<Note>
**Note:** If you select a template while creating an interview, the template interface appears first. From there, you can import questions.
</Note>

For more information, see[📄 Standardizing Interviews Using Templates](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/4ec4b637-6f9b-42b6-83dd-a18c86da04f8).

### Using Code Repositories

Use Code Repositories to access real-world repositories that allow you to assess candidates’ skills in feature building, bug fixing, and other development tasks.

To import a Code Repository:

<Frame>
  <img src="/images/add-questions-to-interview/ba4829d7c77c943b.gif" alt="image" />
</Frame>

1. Select **Code Repositories **from the left panel.
1. Select a repository to view its details and tasks.
1. Select one or more tasks from the repository.
1. Click **Use**.

For more information, see [📄 The Next-Generation of Hiring: Interview Features](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/32521290-82e7-4431-809a-12393c628d44).

### Import Questions from Screen Test

With the **Tests** integration, interviewers can import a candidate’s screening test, including questions and submitted code, directly into an interview session. For more information, see [📄 Import Questions from Screen Test](https://app.notion.com/docs/3147b674-5cbb-4bde-9528-7699bb64b1a7/articles/ecad6d90-60b4-407f-924b-fd330bb3f87a).
