**Note:** If you selected a template when creating the interview, it loads automatically when the interview opens. If you did not select a template, the default interview screen appears. ## Manage templates during an interview

You can update templates directly from the interview interface.

### Add questions to a template

You can update templates during an interview by adding new questions from the Question Library.

To add a question to a template:

1. Open the **Question Library** from the Interview start page, or select **Questions** in the left pane.

  <Frame>
    <img src="/images/use-templates-in-interviews/032126c2536fe5cb.png" alt="image" />
  </Frame>
1. Select a question and click the **(+)** icon.
1. Select the checkbox for the template you want to update. The question is added to the selected template.

  <Frame>
    <img src="/images/use-templates-in-interviews/50a249576cc0ca66.png" alt="image" />
  </Frame>

<Note>
**Note:** You can also select **Create new template** to create a new template from the interview interface. Enter a template name and choose a role. After you create the template, add the question to it using the same **(+)** icon workflow.
</Note>

### Renaming a template

To rename a template during an interview:

1. Open the interview and go to the **Templates** section in the left pane.
1. Select the template you want to rename.
1. Select the edit icon next to the template name.

  <Frame>
    <img src="/images/use-templates-in-interviews/9209c57931d6c2ab.png" alt="image" />
  </Frame>
1. Enter the new name and click the checkmark icon to save it. A success message appears.

  <Frame>
    <img src="/images/use-templates-in-interviews/ce2dd00f8dd57707.png" alt="image" />
  </Frame>

### Removing a question from a template

To remove a question from template during an interview:

1. Open the interview and go to the **Templates** section in the left pane.
1. Select the template you want to update.
1. Select the **(-)** icon for the question you want to remove.

  <Frame>
    <img src="/images/use-templates-in-interviews/24e5a6e1b11fa500.png" alt="image" />
  </Frame>

### Deleting a template

To delete a template during an interview:

1. Open the interview and go to the **Templates** section in the left pane.
1. Select the template you want to delete.
1. Click the **More options** menu (**⋮**) in the upper-right corner and select **Delete Template**.

  <Frame>
    <img src="/images/use-templates-in-interviews/9f0ec8058a04f5db.png" alt="image" />
  </Frame>
1. Click **Delete **in the confirmation dialog.

  <Frame>
    <img src="/images/use-templates-in-interviews/aa2437adcca106b3.png" alt="image" />
  </Frame>
