<Accordion title="**Why does the font appear black in Interview dark mode, and how can I change it?**">
On macOS, the font may appear black in Interview dark mode due to font rendering or browser compatibility issues. Try one of the following solutions.

<Accordion title="**Solution 1: Reinstall the Source Code Pro Font**">
    1. Open the **Font Book** application on your Mac.
    1. In the search bar, type **Source Code Pro**.

      <!-- Unhandled Notion block: numbered_list_item -->

      <!-- Unhandled Notion block: numbered_list_item -->
    1. Right-click the **Source Code Pro** font family and select **Disable “Source Code Pro” Family**.
    1. In the confirmation dialog, click **Disable**.

      <Frame>
        <img src="/images/frequently-asked-questions/2c99389c460c30c8.png" alt="image" />
      </Frame>
    1. Download the **Source Code Pro** font family from the [Google Fonts](https://fonts.google.com/specimen/Source+Code+Pro?query=source%20code%20pro) website.
    1. Install the downloaded fonts using **Font Book**.

      <Frame>
        <img src="/images/frequently-asked-questions/d75acc179223d7a2.png" alt="image" />
      </Frame>

      <Frame>
        <img src="/images/frequently-asked-questions/a9c04339b0eed570.png" alt="image" />
      </Frame>
    1. Refresh the interview page in your browser.

After installation, the font renders correctly in dark mode.
</Accordion>

<Accordion title="**Solution 2: Use the Safari Browser**">
This issue may occur when using third-party browsers on macOS. To resolve it, open the interview in Safari.
</Accordion>
</Accordion>

<Accordion title="**How do I add my own questions to HackerRank for Work?**">
To add questions, you must have a HackerRank for Work account. After you log in, you can create and add questions to your company library. For detailed steps, see [Creating Custom Questions](https://support.hackerrank.com/collections/6791011056-creating-custom-questions).
</Accordion>

<Accordion title="**What should I do if I encounter an issue during my interview?**">
Use a [supported browser](https://candidatesupport.hackerrank.com/articles/8455316478-browser-recommendations-for-hackerrank-interviews) and run the [System Compatibility Check](https://www.hackerrank.com/compatibility) to identify browser, network, or device issues.

If the issue persists, contact support@hackerrank.com and include your interview link.
</Accordion>
